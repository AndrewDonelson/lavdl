# lavdl
Linux Age Verification Distributed Ledger (LAVDL) - A Privacy-preserving age verification for Linux. Cryptographically binds age  attestation to a local user account via an anonymous UUID published to a  decentralized peer-to-peer ledger. Zero PII on the network. Two-byte public  API. Fourth Amendment compliant by design.
